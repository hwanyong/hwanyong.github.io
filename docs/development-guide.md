# 개발 가이드

## 개발 환경 설정

### 필수 요구사항
1. Ruby 설치
2. Bundler 설치
3. Git 설치
4. (선택사항) 코드 에디터 설치 (VS Code 권장)

### 로컬 개발 환경 설정
1. 저장소 복제:
```bash
git clone https://github.com/hwanyong/hwanyong.github.io.git
cd hwanyong.github.io
```

2. 의존성 설치:
```bash
bundle install
```

3. 로컬 서버 시작:
```bash
bundle exec jekyll serve
```

## 프로젝트 구조

### 주요 구성요소
- Jekyll: 정적 사이트 생성
- Sass: 스타일링
- JavaScript: 인터랙티브 기능
- Markdown: 콘텐츠 작성

### 템플릿 시스템
- `_layouts/`의 레이아웃
- `_includes/`의 포함 파일
- `_sass/`의 커스텀 스타일
